# CNPJ Analytics & Compliance Pipeline

## Overview
An end-to-end data engineering project that ingests Brazilian public company data (CNPJ), processes it using a modern data stack, and provides compliance insights through a Data Warehouse.

## Architecture
- **Ingestion:** Python (AsyncIO) & MinIO (S3 Data Lake)
- **Processing:** Apache Spark & DuckDB
- **Transformation:** dbt (Data Build Tool)
- **Storage:** PostgreSQL (Data Warehouse)
- **Orchestration:** Airflow
- **Visualization:** Metabase
